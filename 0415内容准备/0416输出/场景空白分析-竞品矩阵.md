## 二、空白分析

### A. 所有产品都"无"或"弱"的环节 — 最大空白

| 空白环节 | 现状 | 空白原因 |
|---------|------|---------|
| **6.效果追踪** | 全场"无/弱"，无产品做"吃了3个月鱼油→下次体检甘油三酯变化" | 需要长周期数据+前后体检对比+归因分析，技术和数据壁垒高 |
| **7.调整方案** | 全场"无/弱"，无产品做"基于效果反馈→自动建议调整补剂方案" | 需要闭环数据（体检+服用记录+主观感受），没有产品同时拥有这三层数据 |

### B. 无产品串联的环节组合 — 差异化机会

| 未被串联的组合 | 说明 |
|--------------|------|
| **1感知 → 3方案 → 5管理 → 6追踪** | 没有产品从"看懂体检"一路串到"追踪效果"。蚂蚁阿福做到了1→3但跳过了4-7；LemonBox做3→4但缺1和6 |
| **体检报告 + 保健品成分级推荐** | 蚂蚁阿福能读报告但不推荐具体保健品；LemonBox推荐补剂但不读体检报告 |
| **保健品选购 + 服用管理** | LemonBox/Persona只管"卖给你"，Medisafe只管"提醒你吃"，中间无衔接 |
| **效果追踪 + 方案调整** | 全球范围内无产品做到"体检前后对比→自动调整补剂方案"的闭环 |

### C. 结合多维状态的个性化产品

现状：**不存在。** 没有产品同时接入体检数据+日常症状+心情/压力/睡眠来做保健品推荐。蚂蚁阿福有健康日记但不关联保健品；LemonBox有问卷但一次性填完后不动态更新。

## 三、VitaMe 最佳定位建议

### 核心定位：保健品全旅程闭环管理

不做"全覆盖"，锁定**环节1+3+5+6+7**的串联闭环，轻触环节4。

```
[体检报告+症状] → [成分级保健品方案] → [服用管理+提醒] → [效果追踪] → [方案调整]
     ↑ 环节1              环节3              环节5          环节6        环节7
     └──────────────── 数据闭环 ──────────────────────────────────────────┘
```

### 每个环节做到什么程度

| 环节 | VitaMe做法 | 深度 |
|------|-----------|------|
| **1.感知** | 体检报告上传→纵向趋势分析（多年对比）+日常症状/心情记录 | 深做，这是数据入口 |
| **3.方案** | 基于体检异常+症状→推荐到**成分级别**（不是品牌级）。例：甘油三酯偏高→推荐Omega-3(EPA型)，而非笼统说"吃鱼油" | 深做，核心壁垒 |
| **4.选购** | 提供成分比对工具+外部选购链接/视频，不自己卖货 | 轻触即可 |
| **5.管理** | 服用提醒+冲突检查（保健品之间/保健品与药物）+饭前饭后/间隔时间 | 中度，用结构化数据做 |
| **6.追踪** | 下次体检上传后自动对比→可视化"吃了X个月→指标Y的变化" | 深做，全球空白 |
| **7.调整** | 基于追踪结果→AI建议"继续/换/停/加" | 深做，全球空白 |

### 壁垒来源

1. **数据闭环壁垒**：用户一旦上传多年体检报告+记录服用习惯，迁移成本极高。这不是单次问答，是持续积累的健康档案。

2. **医学专业壁垒**：团队有医学检测博士+护士，可以做到"成分级推荐"而非"品牌推荐"。成分级推荐（氧化镁 vs 甘氨酸镁 vs 柠檬酸镁）需要药理学知识，大模型容易幻觉，人工校准是关键。

3. **归因分析壁垒**：环节6-7需要同时拥有"干预前体检+干预措施+干预后体检"三层数据。没有产品同时收集这三层，这就是VitaMe的独占数据。

### 为什么大厂不容易复制

| 大厂 | 为什么不做这个组合 |
|-----|-----------------|
| 蚂蚁阿福 | 3000万月活做的是"广而浅"的AI问诊，不会深入保健品成分级推荐（涉及利益冲突+阿里健康电商利益） |
| 丁香医生 | 医生问诊模式，不做保健品推荐（会被视为"带货"，损害专业形象） |
| LemonBox | 自有品牌DTC模式，推荐结果必然指向自家产品，无法做中立的成分比较 |
| Keep/薄荷 | 基因在运动/饮食，保健品是非核心场景 |
| Medisafe | 药品管理工具，不做健康分析和推荐 |

### 14天Demo优先级

| 优先级 | 功能 | 理由 |
|-------|------|------|
| P0 | 体检报告上传→纵向趋势分析→成分级保健品建议 | 核心故事线，Demo最有冲击力 |
| P0 | 服用提醒+冲突检查 | 实用且好实现 |
| P1 | 效果追踪（模拟数据演示） | 空白最大的差异点，即使用模拟数据也要展示概念 |
| P2 | 选购比价/视频内容 | 锦上添花，Demo阶段可简化 |

### 一句话定位

> **VitaMe：唯一串联"体检趋势→成分级推荐→服用管理→效果追踪→方案调整"的保健品全旅程闭环，让每一粒保健品都有据可查、有效可追。**
