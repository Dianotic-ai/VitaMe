# 产品简报: 健康守护Agent

> 状态：已过期。
> 本文仍以“体检趋势分析 + 微信小程序”作为主路径，已被当前的“补剂安全翻译 Agent + 手机 H5 / 微信内 WebView”方案替代。
> 请优先参考：`docs/product/定位.md`、`docs/product/PRD.md`、`docs/decisions/VitaMe-技术选型-Anthropic-会诊-2026-04-17.md`

## 执行摘要

中国有超过9亿亚健康人群。他们中的很多人每年做体检，但拿到报告后看不懂、不会用；他们想通过保健品改善身体，但面对成分表一头雾水——同样是补镁，氧化镁和甘氨酸镁的吸收率差10倍，价格差一倍，没人告诉他们这些。

**健康守护Agent**是一个AI驱动的健康管理系统。用户上传体检报告，AI自动进行多年趋势分析，结合用户的主观症状和生活习惯，给出个性化的保健品推荐——精确到成分形式、服用时间、禁忌提醒，并通过打卡和情绪鼓励帮助用户真正坚持下去。

现有产品要么做健康科普（丁香医生）、要么做饮食记录（薄荷健康）、要么做运动监测（华为健康），没有一个产品同时解决**"看懂体检→选对保健品→坚持吃"**这三个串联环节。AI Agent的对话式交互让这个串联体验成为可能。

## 问题

### 用户痛苦是真实的

一个30岁的上班族，最近腰痛、眼睛干涩、肠胃不适。他想改善，但遇到了一系列障碍：

1. **体检报告是一次性快照**：每年做体检，但从没人告诉他"你的尿酸连续3年在升，虽然还在正常范围内，但趋势危险"
2. **保健品市场是信息黑洞**：
   - 同一款鱼油，京东和iHerb价差一倍
   - 成分表写的是英文化学名，翻译成中文后搜索结果模糊不清
   - 氧化镁便宜但吸收率只有4%，甘氨酸镁贵但吸收率高——没人解释这些
   - 有结石的人不能随便补钙，服华法林的人不能吃维K——禁忌关系复杂且危险
3. **从"知道"到"做到"断裂**：即使买了保健品，忘记吃、吃错时间、坚持不了是常态

医生不会花时间讲保健品知识。保健品销售只会推销贵的。**用户在健康决策上是孤立无援的。**

## 解决方案

一个统一入口的AI健康管家，核心叙事：**"提前3年发现你的健康拐点"**

### Tab 1: 智能健康Agent（核心对话入口）
- **体检报告纵向分析**：上传多年报告，AI识别趋势变化和关联异常（如MCV↑+B12↓ = 巨幼细胞贫血倾向）
- **症状+报告融合推理**：结合主观不适（腰痛、眼干、肠胃问题）和客观数据，生成综合评估
- **个性化保健品推荐**：精确到成分形式（甘氨酸镁而非氧化镁）、剂量、服用时间、禁忌警告
- **成分冲突检测**（独立传播入口）："你吃的这几样冲突吗？" — 适合社交裂变

### Tab 2: 健康记录与习惯养成
- 服用打卡 + 饮水监控 + 久坐提醒
- 情绪鼓励和阶段性反馈
- 微量元素每日达标追踪

### Tab 3: 选购决策支持
- 保健品成分标签中英文解析
- 合规性标注（蓝帽子认证状态）
- 优质内容聚合（专业视频 + 图文）
- 跨平台比价（后续迭代）

## 差异化优势

| 维度 | 现有产品 | 健康守护Agent |
|------|---------|--------------|
| 体检报告 | 单次解读 | **多年纵向趋势分析** |
| 保健品推荐 | 按品牌/品类 | **按成分形式+个人禁忌** |
| 成分信息 | 英文/学术级 | **中文循证解析，小白可读** |
| 交互方式 | 表单/文章 | **AI对话，多轮推理** |
| 行为闭环 | 止步于信息 | **打卡+提醒+鼓励** |

**核心壁垒**：体检趋势分析 × 保健品循证推荐 × 行为闭环，三者串联形成完整价值链。市场上无同类产品。

## 目标用户

### 黑客松Demo聚焦人群
- **亚健康年轻人（25-35岁）**：久坐办公，身体开始发信号（腰痛、眼干、肠胃不适），有改善意愿但不知从何入手。Demo只为这一个画像优化体验。

### 后续扩展人群
- **中老年人（45-65岁）**：体检报告积累多年，有保健品服用习惯但缺乏科学指导
- **宝妈**：为家庭成员（老人、孩子）管理健康 → 未来"家庭组"功能
- **健身人群**：补剂选购需求明确，对成分形式和剂量敏感

### "啊哈时刻"
用户上传体检报告后，Agent 说："你的尿酸从2020年的340持续升到2025年的410，虽然还在正常范围，但按这个趋势2年内可能超标。建议你减少高嘌呤饮食，同时可以考虑补充维C（500mg/天）辅助降尿酸——注意选择缓释型维C，不要一次性大剂量，因为你有结石病史。"

这种**个性化、有理有据、考虑禁忌**的建议，是用户从任何现有产品都得不到的。

## 范围

### MVP（黑客松版本）— 必须有
- AI对话式健康评估（体检指标手动录入 + 症状描述）
- Top 50高频成分知识库（覆盖80%场景）
- 成分形式差异解析 + 禁忌安全规则引擎
- 成分冲突检测独立入口（裂变传播点）
- 免责声明系统（UI层强制展示，每次输出附带）
- 首次使用即时价值：单次报告解读 + 同龄人指标对比（解决冷启动问题）

### MVP — 明确排除
- 体检报告OCR（手动录入优先）
- 跨平台比价爬虫
- 穿戴设备接入
- 打卡功能（Demo阶段极简化或砍掉，聚焦"体检→选品"核心链路）
- 家庭组/社交功能

### 后续迭代
- 体检报告照片OCR解析
- 打卡与习惯养成完整功能
- 京东/iHerb比价
- 家庭组（宝妈管全家报告）
- B端企业团检合作
- 智能手表数据接入
- 中医体质辨识融合模块

## 商业模式

### MVP阶段（零成本验证）
- 基础功能免费
- 保健品推荐导购佣金（CPS 5-15%）
- **注意**：佣金与推荐算法须解耦，推荐基于循证而非佣金率；UI透明披露合作关系

### 长期路径
- 会员订阅 ¥19.9/月（解锁深度趋势分析和个性化方案）
- B端品牌精准营销 + 真实服用依从性报告（品牌方高付费意愿）
- 企业团检后健康管理
- 健康保险联动

## 愿景

如果成功，健康守护Agent将成为每个人的**私人健康顾问**——它了解你过去十年的身体变化，知道你正在吃什么保健品，会在你忘记吃药时温柔提醒，会在发现危险趋势时及时预警。

2-3年后，接入穿戴设备实时数据后，它不仅分析过去，还能预测未来——从"被动体检"到"主动健康管理"的范式转变。

## 关键风险与缓解

| 风险 | 级别 | 缓解措施 |
|------|------|---------|
| AI幻觉导致错误建议 | 🔴极高 | RAG架构+禁忌硬编码+置信度阈值，低置信度回退"建议咨询医生" |
| 医疗建议法律风险 | 🔴极高 | 定位"健康信息服务"非"医疗服务"；免责声明UI强制展示；措辞合规审查 |
| "循证"措辞风险 | 🔴高 | 明确数据源和证据分级；证据不足时标注"有限证据"而非断言；避免与医疗诊断混淆 |
| 健康数据隐私 | 🔴高 | 遵循《个人信息保护法》，敏感数据单独授权+加密+数据最小化 |
| CPS佣金与推荐中立性冲突 | 🟡中 | 推荐算法与佣金解耦；透明披露合作关系 |
| 成分数据不完整 | 🟡中 | 优先Top 50成分，未收录标注"数据库暂未收录" |
| 黑客松范围风险 | 🟡中 | Demo明确只演示核心链路，部分数据用mock |

## 技术路径

- **框架**：Hermes Agent（https://github.com/NousResearch/hermes-agent）
- **初期形态**：微信小程序
- **成分数据源**：DSLD公开API + 手工整理Top 50知识库
- **体检解析**：MVP阶段手动录入，后续接入OCR
- **安全架构**：系统提示词嵌入硬性安全规则 + 输出后处理过滤
