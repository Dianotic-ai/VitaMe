# 商业化深度分析

## 推荐变现组合（Phase 1）
- 单次报告付费 ¥9.9（引流转化）
- CPS导购佣金 5-15%（自然变现，推荐与佣金解耦）
- 订阅制 ¥19.9/月 or ¥168/年（核心营收）

## 免费vs付费边界
- 免费：1份报告基础解读 + 3次AI对话/天 + 基础打卡
- 付费：无限报告+纵向趋势+循证方案+无限对话+月度简报

## 流量闭环
- 获客：体检季投放 + 搜一搜截流 + 小红书种草
- 激活：90秒内生成趋势图+风险提示（Aha Moment）
- 留存：月度健康简报 + 指标倒计时游戏化
- 裂变：健康趋势卡分享 + 组队打卡 + 年度健康报告
- 转化：免费上传→趋势预警→付费深度分析→CPS购买→订阅

## 冷启动（前100用户）
- Week1: 朋友圈+私聊刚体检的朋友(0-20人)
- Week2: 小红书体验帖+知乎长尾词(20-50人)
- Week3: 种子用户邀请奖励(50-100人)

## 竞争壁垒
- 短期：三合一闭环体验
- 中期：用户纵向数据积累，迁移成本高
- 长期：循证推荐口碑+社区+可能的资质
