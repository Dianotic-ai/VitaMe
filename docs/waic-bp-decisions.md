### 决策 #3 — 商业模式（P12）⭐

**为什么需要**：杨露茜 + 4 个投资人最关心。**必须明确选 1 个主路径**，不能贪心 4 个全要。

**4 条候选**：

| 路径 | 谁付钱 | 优势 | 劣势 |
|---|---|---|---|
| **A. C 端订阅** | 普通用户 ¥9.9-19.9/月 | 启动快 / 规模大 / mom 用户付费意愿强 | 获客成本高 |
| **B. B 端 SaaS** | 连锁药店 / 保健品品牌 | 客单高 / 数据壁垒 / 复购稳 | 销售周期长 / 需要 B 端关系 |
| **C. 抽佣电商** | 用户买保健品时分佣 | 现金流快 | 公信力被电商绑架 |
| **D. 数据合作** | 药企 / 保险公司 | 高毛利 | 合规复杂 / 隐私冲突 |

**我的判断**：**主推 A，备 B**。理由：
- A 的 mom 用户切入跟杨露茜投资品味（消费 + 社区）一致
- B 是中长期升级路径（先有 C 端用户量再谈 B）
- C 损害医疗公信力（合规 + 体验风险），不推
- D 跟北极星 §9.8 隐私底线冲突，5 年内不做
