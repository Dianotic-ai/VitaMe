## A. 药物 × 补剂冲突(10 条)

### Q1
- **Q 原文**:在吃 Eva(抗抑郁药)期间可以吃鱼油+维 D+镁+益生菌+维 B+维 C 吗?会不会肝脏代谢不了?
- **来源**:真实评论(湖南 2025-08-07)
- **场景类**:FR-3 + 药物×补剂交互 + 器官代谢焦虑
- **期望等级**:黄
- **VitaMe 输出概要**:
  - 黄:SSRI 与高剂量鱼油(>3g/天)需警惕 5-羟色胺协同风险
  - 翻译:"肝代谢"不是主要风险,5-羟色胺协同才是
  - 规避:鱼油 1-2g/天;维 B6 ≤50mg/天;分开服用更稳

### Q2
- **Q 原文**:我爸在吃降压药(氨氯地平),可以吃鱼油吗?
- **来源**:合理补充(PRD §8 用户故事 2)
- **场景类**:FR-3 + 家人场景 J3
- **期望等级**:黄
- **VitaMe 输出概要**:
  - 黄:鱼油有轻度降压作用,叠加降压药可能低血压
  - 翻译:不是"不能吃",是要监测血压
  - 建议:起始 1g/天,1-2 周复测血压

### Q3
- **Q 原文**:吃他汀降脂药,还需要补辅酶 Q10 吗?
- **来源**:合理补充(慢病高频)
- **场景类**:FR-3 + 补充性建议
- **期望等级**:绿
- **VitaMe 输出概要**:
  - 绿:他汀降低内源性 Q10 水平,补充可缓解肌肉不适
  - 翻译:这是"补偿用药耗损",不是冲突
  - 建议:100-200mg/天;和他汀分开 4 小时

### Q4
- **Q 原文**:妈妈在吃华法林抗凝,她朋友推荐的辅酶 Q10 能吃吗?
- **来源**:合理补充(家人 + 高风险药)
- **场景类**:FR-3 + 家人场景 + 高风险药物
- **期望等级**:红
- **VitaMe 输出概要**:
  - 红:Q10 有 Vitamin K 样作用,可能降低华法林抗凝效果
  - 翻译:不是玄学,是药物抵抗机制
  - 建议:开始/停用前必须咨询医生并监测 INR

### Q5
- **Q 原文**:甲亢在吃甲巯咪唑,能同时补碘吗?
- **来源**:合理补充(内分泌高频)
- **场景类**:FR-3 + 内分泌
- **期望等级**:红
- **VitaMe 输出概要**:
  - 红:甲亢患者额外补碘会加重甲亢,可能诱发甲亢危象
  - 翻译:碘是甲状腺激素"原料",病理状态下补碘=火上浇油
  - 建议:禁用含碘补剂(海藻/紫菜/螺旋藻提取物);用药期间必须遵医嘱

### Q6
- **Q 原文**:吃左氧氟沙星消炎,能吃钙镁锌复合片吗?
- **来源**:合理补充(抗生素高频)
- **场景类**:FR-3 + 吸收抑制
- **期望等级**:黄
- **VitaMe 输出概要**:
  - 黄:二价/三价金属离子(Ca/Mg/Zn/Fe)与喹诺酮类螯合,抗生素吸收率可下降 50%+
  - 翻译:不是"冲突",是"你吃了药等于没吃"
  - 建议:抗生素和含金属补剂间隔至少 2 小时,优先隔开 4 小时

### Q7
- **Q 原文**:吃安眠药(艾司唑仑),还能吃褪黑素吗?
- **来源**:合理补充(精神类)
- **场景类**:FR-3 + 中枢抑制叠加
- **期望等级**:黄
- **VitaMe 输出概要**:
  - 黄:苯二氮䓬类 + 褪黑素中枢抑制叠加,次日嗜睡/反应迟缓
  - 翻译:都是"关灯"的药,两个一起关太狠
  - 建议:二选一;如必须同用,褪黑素减至 0.5-1mg,并咨询精神科医生

### Q8
- **Q 原文**:吃避孕药能同时吃 St.John's Wort(圣约翰草)吗?
- **来源**:合理补充(草本类陷阱)
- **场景类**:FR-3 + 草本药物交互
- **期望等级**:红
- **VitaMe 输出概要**:
  - 红:圣约翰草诱导 CYP3A4 酶,加速避孕药代谢,导致避孕失败
  - 翻译:不是"效果减弱",是"可能怀孕"
  - 建议:避孕药期间禁用圣约翰草

### Q9
- **Q 原文**:吃布洛芬缓解痛经,能一起吃维 E 吗?都说维 E 活血
- **来源**:合理补充(凝血风险)
- **场景类**:FR-3 + 出血风险叠加
- **期望等级**:黄
- **VitaMe 输出概要**:
  - 黄:NSAIDs + 高剂量维 E(>400IU/天)理论上增加出血风险,经期背景下更需警惕
  - 翻译:两个都有抗凝倾向,叠加会让经量更大
  - 建议:维 E ≤200IU/天;经期不额外加量;必要时布洛芬饭后服

### Q10
- **Q 原文**:吃左甲状腺素(优甲乐),能吃钙片吗?
- **来源**:真实评论模式泛化
- **场景类**:FR-3 + 吸收抑制(时序型)
- **期望等级**:黄
- **VitaMe 输出概要**:
  - 黄:钙与优甲乐同服显著降低其吸收(高达 30%+)
  - 翻译:不是冲突,是"顺序问题"
  - 建议:优甲乐晨起空腹;钙片晚餐后或午餐后,间隔≥4 小时

## B. 病史 × 补剂冲突(10 条)

### Q11
- **Q 原文**:家里老人有肝炎+脂肪肝,apoe4 基因,可以吃鱼油吗?什么样的鱼油?
- **来源**:真实评论(湖南 01-04)
- **场景类**:FR-3 + 家人 + 基因精准
- **期望等级**:黄
- **VitaMe 输出概要**:
  - 黄:肝炎活动期需先看 AST/ALT;apoe4 对 DHA 代谢有差异
  - 追问(自动化):肝功当前值?凝血功能正常吗?
  - 建议:优先 EPA 主(EPA:DHA≥2:1);1g/天起;3 个月复查肝功

### Q12
- **Q 原文**:有肾结石,可以吃维 D 和钙片吗?维 C 呢?
- **来源**:合理补充(PRD §8 用户故事 1)
- **场景类**:FR-3 + 病史交叉
- **期望等级**:黄
- **VitaMe 输出概要**:
  - 黄:维 D 促钙吸收需控制总钙;维 C >1g/天增加草酸钙结石风险
  - 翻译:不是不能补,是盯"剂量上限"
  - 建议:维 D 600-1000IU/天;钙优先食补;维 C ≤500mg/天

### Q13
- **Q 原文**:胃溃疡体质,听说镁能助眠但容易腹泻,吃什么形式的?
- **来源**:合理补充(0417 Hero 示例)
- **场景类**:FR-3 + FR-4 形式翻译
- **期望等级**:黄
- **VitaMe 输出概要**:
  - 黄:氧化镁吸收差且易腹泻,胃肠敏感不合适
  - 翻译:镁形式差异 = 吸收率 + 刺激性
  - 建议:甘氨酸镁或苏糖酸镁,睡前 200mg

### Q14
- **Q 原文**:高血压+糖尿病,在吃二甲双胍和降压药,可以吃铬和肉桂粉吗?
- **来源**:合理补充(多重慢病)
- **场景类**:FR-3 + 多重慢病
- **期望等级**:黄
- **VitaMe 输出概要**:
  - 黄:铬/肉桂有降糖作用,与二甲双胍叠加可能低血糖
  - 建议:起始低剂量;监测血糖;咨询医生是否调整用药

### Q15
- **Q 原文**:甲减在吃优甲乐,有人说不能和钙片一起吃?
- **来源**:合理补充(甲状腺高频)
- **场景类**:FR-3 + 时序问题
- **期望等级**:黄
- **VitaMe 输出概要**:
  - 黄:钙抑制优甲乐吸收,需间隔 4 小时
  - 翻译:不是不能一起补,是"顺序"问题
  - 建议:优甲乐晨起空腹;钙片晚餐后

### Q16
- **Q 原文**:我有桥本甲状腺炎,能吃硒酵母吗?
- **来源**:合理补充(自免高频)
- **场景类**:FR-3 + 自身免疫
- **期望等级**:绿
- **VitaMe 输出概要**:
  - 绿:循证支持硒(200μg/天)可降低桥本患者抗体(TPOAb)
  - 翻译:这是"少数有证据的桥本辅助补剂"
  - 建议:硒酵母 100-200μg/天;连续 3-6 个月复查抗体

### Q17
- **Q 原文**:痛风尿酸高,听说不能吃烟酸,维 B 族还能吃吗?
- **来源**:合理补充(代谢疾病)
- **场景类**:FR-3 + 代谢病
- **期望等级**:黄
- **VitaMe 输出概要**:
  - 黄:高剂量烟酸(>1g/天)升高尿酸;复合 B 中的烟酸剂量一般较低(<50mg)通常安全
  - 翻译:问题不在"维 B",在"烟酸剂量"
  - 建议:选不含烟酸或烟酸 <50mg/天的 B 族;痛风急性期暂停

### Q18
- **Q 原文**:我有贫血(缺铁),维 C 补多少帮助铁吸收?
- **来源**:合理补充(贫血高频)
- **场景类**:FR-3 + 协同补充
- **期望等级**:绿
- **VitaMe 输出概要**:
  - 绿:维 C 100-200mg 与铁剂同服可显著提升非血红素铁吸收
  - 翻译:维 C 把三价铁还原成更易吸收的二价铁
  - 建议:铁剂 + 维 C 100-200mg 空腹同服;避开钙/茶/咖啡 2 小时

### Q19
- **Q 原文**:乳糖不耐受的人,能吃含乳糖的益生菌胶囊吗?
- **来源**:合理补充(食物不耐受)
- **场景类**:FR-3 + 过敏/不耐受
- **期望等级**:灰
- **VitaMe 输出概要**:
  - 灰:大部分胶囊型益生菌乳糖含量极低,通常可耐受;需看 Supplement Facts 是否含乳糖作为赋形剂
  - 建议:优先无乳糖标注的产品;严重者选酪酸梭菌等无奶源菌株

### Q20
- **Q 原文**:有轻度脂肪肝,水飞蓟能吃吗?
- **来源**:合理补充(肝病草本)
- **场景类**:FR-3 + 肝病辅助
- **期望等级**:绿
- **VitaMe 输出概要**:
  - 绿:水飞蓟宾对非酒精性脂肪肝有循证支持(改善 ALT/AST)
  - 翻译:不是"治病",是辅助+生活方式才是主力
  - 建议:140mg BID 或 200mg QD;3 个月复查肝功

## C. 多补剂组合 × 时间安排(10 条)

### Q21
- **Q 原文**:鱼油、维 D、D3、镁、益生菌、维 B、维 C 都吃,每天什么时间吃?需要间隔多久?
- **来源**:真实评论(贵州 2025-08-21,博主回复 15 赞)
- **场景类**:FR-3 + FR-4 时间表
- **期望等级**:绿
- **VitaMe 输出概要**:
  - 绿:大部分无需严格间隔
  - 表:早空腹→维B/C/益生菌;随餐→D3/鱼油;睡前→镁
  - 翻译:水溶性走尿代谢;脂溶性需脂肪吸收

### Q22
- **Q 原文**:一天吃 6 种补剂,分几次吃合适?能不能一把全吃?
- **来源**:合理补充
- **场景类**:FR-4 时间分配
- **期望等级**:灰
- **VitaMe 输出概要**:
  - 灰:无明显禁忌,分次效果更好
  - 建议:至少分 2 次(早/晚);避免脂溶性和高剂量钙镁同服

### Q23
- **Q 原文**:补剂要不要一起补?还是先补一种看效果再加下一种?
- **来源**:合理补充(决策辅助)
- **场景类**:FR-4 + 决策辅助
- **期望等级**:灰
- **VitaMe 输出概要**:
  - 灰:症状明确指向某种(如睡眠差→镁)先单一 2-4 周;一般保健可组合起步
  - 建议:单一 2-4 周观察,记录反应

### Q24
- **Q 原文**:钙和镁可以一起吃吗?听说会互相抵消
- **来源**:合理补充(常见误解)
- **场景类**:FR-4 + 误解澄清
- **期望等级**:灰
- **VitaMe 输出概要**:
  - 灰:常规剂量下(Ca<1000mg/Mg<400mg)同服基本不抵消;但高剂量钙会竞争性抑制镁吸收
  - 翻译:"抵消"说法被放大了,关键是总量
  - 建议:钙镁比例 2:1 或 1:1 都可接受;或分两次服用

### Q25
- **Q 原文**:铁和钙能一起吃吗?
- **来源**:合理补充(常见配伍)
- **场景类**:FR-4 + 吸收干扰
- **期望等级**:黄
- **VitaMe 输出概要**:
  - 黄:钙显著抑制非血红素铁吸收(同服可降 50%+)
  - 翻译:两个都抢同一个"门"(DMT1 转运蛋白)
  - 建议:间隔至少 2 小时;铁空腹+维C,钙随餐

### Q26
- **Q 原文**:锌和铜要不要一起补?听说锌会让铜缺乏
- **来源**:合理补充(长期补充风险)
- **场景类**:FR-4 + 长期配伍
- **期望等级**:黄
- **VitaMe 输出概要**:
  - 黄:长期高剂量锌(>40mg/天>3 个月)可诱导金属硫蛋白,降低铜吸收
  - 建议:长期补锌时按 Zn:Cu = 10:1-15:1 配比;或定期血清铜复查

### Q27
- **Q 原文**:维 D 和维 K2 一定要一起吃吗?
- **来源**:合理补充(网红组合)
- **场景类**:FR-4 + 协同配伍
- **期望等级**:灰
- **VitaMe 输出概要**:
  - 灰:维 K2 引导钙入骨而非血管,理论上与维 D 协同;但证据等级中等,不是"必须"
  - 翻译:"钙搬运工"组合是合理的,不是必要的
  - 建议:维 D 3000IU + K2 MK-7 100μg 为常见配比;也可单独维 D

### Q28
- **Q 原文**:益生菌要空腹吃还是饭后吃?跟抗生素怎么错开?
- **来源**:合理补充(益生菌高频)
- **场景类**:FR-4 + 菌株时序
- **期望等级**:绿
- **VitaMe 输出概要**:
  - 绿:多数菌株饭前 30 分钟或随餐(低胃酸时)存活率更高
  - 抗生素:与抗生素间隔至少 2-3 小时
  - 建议:睡前或早餐前;抗生素疗程结束后继续补 2-4 周

### Q29
- **Q 原文**:铁剂什么时候吃?早上空腹对胃刺激大,饭后吸收又差
- **来源**:合理补充(铁剂高频)
- **场景类**:FR-4 + 吸收 vs 耐受
- **期望等级**:灰
- **VitaMe 输出概要**:
  - 灰:空腹吸收最好但胃刺激;新证据:隔天一次反而吸收率更高
  - 翻译:每天吃身体会自我保护(hepcidin 上调)反而不吸收
  - 建议:隔天空腹+维C;胃受刺激改为餐后

### Q30
- **Q 原文**:运动前后的营养补充怎么安排?蛋白粉、BCAA、肌酸、维生素该怎么搭?
- **来源**:合理补充(健身人群)
- **场景类**:FR-4 + 运动营养
- **期望等级**:绿
