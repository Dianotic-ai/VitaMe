# VitaMe — 品牌视觉规范

## 1. 视觉定位

VitaMe 的视觉目标不是“医疗工具感”或“保健品电商感”，而是：

- 温柔可信：像一个谨慎、懂边界的健康助手。
- 高级克制：少装饰、强留白、低噪音。
- 有机生命力：用“种子发芽”表达辨别之后的生长。
- 当代感：不是复古草本、不是赛博医疗、不是卡通 wellness。

首页、产品页、Demo 页和 AI 生成视觉都必须先满足这一层判断，再考虑转化效率。

## 2. Core Metaphor

核心隐喻：**种子正在发芽**。

它对应 VitaMe 的产品判断：

- 用户不是来买更多补剂，而是想知道“我现在吃的这些东西是否安全、是否适合我”。
- 产品不是强干预医生，也不是导购，而是帮助用户辨别、保存、复查。
- “发芽”表示在复杂健康信息里找回秩序，而不是给用户制造新的健康焦虑。

这个隐喻应贯穿视觉、文案、图形语言和交互节奏。

## 3. Style Keywords

必须同时满足：

- Organic
- Minimal
- Quiet
- Gentle
- Thoughtful
- Airy
- Human
- Warm
- Editorial
- Hand-drawn

禁止滑向：

- Cyberpunk
- SaaS dashboard heavy
- Neon gradient
- Over-decorated
- Cute / cartoonish
- Luxury fashion arrogance
- Wellness cliché
- 保健品促销页

## 4. Color System

| 用途 | 色值 | 名称 | 规则 |
|---|---|---|---|
| 主色 | `#2D5A3D` | 深林绿 | 标题点缀、按钮、芽叶、重点视觉 |
| 次色 | `#8B6B4A` | 种子棕 | 辅助图形、线条、局部卡片 |
| 点缀 | `#4A90B8` | 溪水蓝 | 少量用于提示、过滤、流动、状态 |
| 背景 | `#FAF7F2` | 羊脂白 | 大面积背景和温暖留白 |
| 正文 | `#1C1C1C` | 墨黑 | 正文、标题、深背景区 |

配色规则：

- 页面 70% 以上应由浅背景和留白构成。
- 主色与次色构成骨架，蓝色只能小面积点缀。
- 不使用高饱和撞色。
- 不用炫目渐变作为主体。

## 5. Typography

排版方向：

- 标题应像“洞察”，不是“广告语”。
- 标题字体可有 editorial 气质，正文必须清晰、克制、可读。
- 字号层级要拉开，但不能靠粗黑大字制造压迫感。
- 行距宽松，每个 section 尽量只保留一个视觉主句。

禁止：

- “革命性”“颠覆”“暴涨”等夸张商业话术。
- 促销式 CTA，如“立即抢购”“限时领取”。
- 过密信息墙。

## 6. Illustration Language

参考方向：

- Paul Klee：几何但有机，轻盈但有哲思。
- Saul Steinberg：极简线条、一笔成形。
- 当代植物手绘：细线条、叶脉、种子、豆荚、发芽结构。

可用元素：

- 种子
- 裂开的种皮
- 细芽
- 豆荚
- 根系
- 叶脉
- 水流线

禁止：

- 拟人卡通。
- 复杂写实植物堆叠。
- 大片纹理噪点。
- 过强装饰边框。
- 让插画变成“装饰”，而不是“生长中的结构”。

## 7. Layout Principles

页面结构优先采用：

- 大留白。
- 卡片式分区。
- 非拥挤模块节奏。
- 轻微错位的有机布局。
- 左文右图或中轴 Hero，避免密集 dashboard。

组件原则：

- CTA 克制可信，优先表达“开始判断 / 查看风险 / 保存结果”。
- 风险标签必须遵守 `docs/product/风险判定矩阵.md` 和 `docs/product/信任与Claims边界.md`。
- AI 生成内容同屏必须可见免责声明或边界提示。
- 视觉不能暗示“诊断、治疗、治愈、推荐购买”。

## 9. AI 出图 Prompt 模板

```text
Create a quiet, premium, organic minimalist homepage illustration for VitaMe,
a supplement safety and personal health-context product.

Core metaphor: a seed gently sprouting, representing careful judgment, safety,
and personal growth after understanding risk.

Style: warm editorial minimalism, hand-drawn botanical linework, airy layout,
soft cream background, deep forest green, seed brown, tiny stream-blue accents.

Avoid: cyberpunk, neon gradients, 3D mascot, cute cartoon, medical hospital cliché,
wellness stock art, supplement sales imagery, over-decoration.
```

## 10. Review Checklist

上线或生成任何 UI / 插画前，按 10 项检查。通过标准：至少 9 项为 yes。

1. 第一眼是否像“安静、高级、有生命力”的品牌，而不是工具网站？
2. 是否保留足够留白，页面大面积浅色和空白？
3. 是否有明确的“种子发芽”隐喻或生长结构？
4. 主角插画是否轻盈克制，而不是装饰堆砌？
5. 配角装饰元素是否控制在少量颜色和少量图形内？
6. 色彩是否落在本规范 token 内？
7. 标题是否像洞察，而不是广告语？
8. CTA 是否克制可信，不促销、不制造焦虑？
9. 是否同时具备人文感与当代感？
10. 任何 AI 生成内容是否同屏可见信任边界或免责声明？
