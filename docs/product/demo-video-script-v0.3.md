# VitaMe — 90 秒 Demo 视频脚本

> **总时长**：90 秒
> **设备**：iPhone（中等屏，14 / 15 系列），微信内置浏览器打开 https://vitame.live
> **配音**：女声中文 / 中性偏亲和 / 语速 ~280 字/分（不要播音腔）
> **背景音乐**：克制的 lo-fi 钢琴，30% 音量（不抢配音）
> **字幕**：中文，下方居中，强调关键词**加粗**
> **风格**：所有截屏都是**真机录屏**，不要 mockup / 不要快进刻意作秀

## 后期制作 checklist

### 录屏阶段
- [ ] iPhone 14/15 系列，关闹钟/通知
- [ ] WiFi 稳定（vitame.live 在 Vercel SFO，国内访问 5-7s 是好状态，超 15s 重录）
- [ ] 屏幕亮度调到 80%，避免反光
- [ ] 提前在 vitame.live 把 3 个档案（"我自己"+"妈妈"+"爸爸"）填好
  - **我自己**：长期用药"华法林"
  - **妈妈**：65 岁、女、长期用药"硝苯地平"（降压药）
  - **爸爸**：可以空着 demo 不用
- [ ] 录屏用 iPhone 自带"屏幕录制"，输出 1080p 60fps

### 剪辑阶段
- [ ] 配音用 ElevenLabs 中文声 / 自己用 iPhone 录（安静房间 + 麦克风）
- [ ] 字幕用 Final Cut / Premiere / CapCut
- [ ] 关键转场处加 0.3s 缓动（避免突兀切换）
- [ ] 流式输出片段保持原速（不要快进，那个慢慢蹦字才是产品价值）
- [ ] 总时长严格 90 秒（PPT 嵌入有缓冲，超就要重剪）

### 输出
- [ ] mp4 格式，h.264，1080p
- [ ] 文件大小 < 50MB（PPT 嵌入流畅）
- [ ] 备一份 720p 低码率 fallback（现场投屏带宽不稳时用）
- [ ] 上传一份到 B 站不公开 / Google Drive，二维码备一个
