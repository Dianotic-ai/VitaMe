## 视觉规范（PPT 制作时遵守）

- **字体**：标题 思源宋体 / 正文 思源黑体（跟产品 UI 一致）
- **色板**：跟 vitame.live 同 — 暖白底 (#FAF8F4) + emerald 主色 (#059669) + 警示用 amber-700
- **不用**：渐变彩虹 / 3D 图标 / 拟物化 / icon 堆砌
- **每片字数 ≤ 25 字**（除非数据卡片）
- **截图来源**：用 vitame.live 真机录屏的截屏，不要 mockup（评委能看出区别）
- **Speaker notes 写在备注页**，不展示给评委

## 排练建议

1. **第一遍掐时间**：每片单独练，确认不超时
2. **第二遍连贯**：从片 1 走到片 8，看转场是否丝滑
3. **第三遍带视频**：在片 4 真的播 90s 视频，看观众回血时间够不够
4. **第四遍模拟 Q&A**：让搭档随机问 3 个问题（数据来源 / 商业模式 / 团队），确认 30s 内回答得了
5. **现场前 1 小时**：在投屏设备上过一遍，确认色板没色偏

## 评委可能问的 5 个问题（提前准备答案）

1. **"你们跟丁香医生有什么不一样？"**
   → 丁香是医生咨询，我们是补剂顾问；丁香按次付费，我们对话免费 + 长期记忆

2. **"AI 给的建议出错了谁担责？"**
   → 7 条合规底线全部带 disclaimer + DemoBanner + audit log；高危组合硬路由强制"咨询医生"，不让 LLM 自由发挥

3. **"你们的数据从哪来？"**
   → 8 个公开权威源（NIH ODS / LPI / 中国营养学会 DRIs / SUPP.AI / DSLD / TGA / 日本机能性 / 蓝帽子），全部带 sourceRefs，可追溯

4. **"商业模式？"**
   → P0 不收费验证产品力。P1 加品牌推荐 CPS / 订阅"健康顾问 Pro"（无限对话 + 长期跟踪）

5. **"为什么不做 App 直接做 H5？"**
   → 微信 WebView 适配中老年用户最友好；妈妈这一波用户装 App 阻力大，扫码即用
