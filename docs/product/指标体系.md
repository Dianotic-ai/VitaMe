# VitaMe — 指标体系

## 1. North Star

长期 North Star：**Agent Learning Loop Completion**

定义：用户完成一次安全判断，并在后续通过提醒或反馈产生可用于 Memory 的有效上下文。

P0 暂用代理指标：**Qualified Safety Decision**。

## 2. Phase Metrics

| Phase | North Star Proxy | Definition |
|---|---|---|
| P0 | Qualified Safety Decision | 完成查询并看到可解释风险结果 |
| P1 | Feedback Loop Completion | 设置提醒并完成一次有效反馈 |
| P2 | Confirmed Agent Learning | Hermit 观察被用户确认或纠错 |

## 3. Input Metrics

| Metric | Definition | Target |
|---|---|---|
| Query Start Rate | 访问首页后发起查询比例 | P0 观测 |
| Query Completion Rate | 发起后看到结果页比例 | >= 80% |
| Save Rate | 首次结果保存比例 | >= 40% |
| Reminder Opt-in Rate | 保存后确认提醒比例 | P1 观测 |
| Feedback Response Rate | 提醒后反馈比例 | >= 40% |
| Memory Density | 30 天有效 Memory 数 | P2 观测 |

## 4. Health Metrics

| Metric | Guardrail |
|---|---|
| Gray Mislabel Rate | 灰色不得被写成安全 |
| Disclaimer Coverage | 结果页、反馈页、Hermit 观察 100% 有边界提示 |
| Banned Phrase Hit | 禁词命中必须被替换或 fallback |
| Reminder Fatigue | 连续跳过或关闭提醒必须触发降频 |
| Hermit Rejection Rate | 高拒绝率说明观察质量差 |

## 5. Business Metrics

P0 不优化收入。P1/P2 后才评估：

- 7 日/30 日留存。
- 家庭成员创建率。
- 分享率。
- 订阅转化意愿。

## 6. Review Cadence

- Daily：API error、主链完成率、合规错误。
- Weekly：QSD、保存率、反馈率。
- Monthly：Memory 密度、Hermit 确认率、留存。
- Quarterly：North Star 是否仍代表核心价值。
