# VitaMe — 竞品参考素材

## 用途

这份文档只承担素材索引职责：帮助 PM、设计和 Agent 找到已保留的竞品截图，作为 UI 气质、信息架构和市场表达参考。

它不是当前产品定位事实源。竞品判断以 `docs/research/场景空白分析-竞品矩阵.md`、`docs/research/竞品调研报告.md` 和 `docs/product/当前判断.md` 为准。

## 使用规则

- 只把这些图片当作参考素材，不作为 VitaMe 当前 UI 规范。
- 如果从截图里抽取产品 insight，必须写入对应 current 文档，而不是让图片承担隐性知识。
- 如果素材不再服务当前 P0 判断，应移入 `docs/_archive/`。
