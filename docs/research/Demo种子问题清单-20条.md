## 使用说明

**每条包含 5 个字段**：
- **Q 原文**：用户原始或语义等效提问

## A. 药物 × 补剂冲突（4 条）

### Q1

- **Q 原文**：在吃 Eva（抗抑郁药）期间可以吃鱼油+维 D+镁+益生菌+维 B+维 C 吗？会不会肝脏代谢不了？
- **来源**：真实评论（第 1 图，湖南 2025-08-07）
- **场景类**：FR-3（安全判断）+ 药物×补剂交互 + 器官代谢焦虑
- **期望等级**：黄色注意
- **VitaMe 输出概要**：
  - 黄：SSRI 类抗抑郁药与高剂量鱼油（>3g/天）需警惕 5-羟色胺协同风险
  - 翻译："肝代谢"不是主要风险，5-羟色胺协同才是
  - 规避：鱼油 1-2g/天；维 B6 不超过 50mg/天；维 B 和 SSRI 分开服用更稳

### Q2

- **Q 原文**：我爸在吃降压药（氨氯地平），可以吃鱼油吗？
- **来源**：合理补充（对应 PRD §8 用户故事 2）
- **场景类**：FR-3 + 家人场景（J3）
- **期望等级**：黄色注意
- **VitaMe 输出概要**：
  - 黄：鱼油有轻度降压作用，与降压药叠加可能出现低血压
  - 翻译：不是"不能吃"，是要监测血压
  - 建议：起始 1g/天，服用 1-2 周复测血压

### Q3

- **Q 原文**：吃他汀降脂药，还需要补辅酶 Q10 吗？
- **来源**：合理补充（慢病高频）
- **场景类**：FR-3 + 补充性建议
- **期望等级**：绿色（推荐补）
- **VitaMe 输出概要**：
  - 绿：他汀会降低内源性辅酶 Q10 水平，补充可缓解肌肉不适
  - 翻译：这是"补偿用药耗损"场景，不是冲突
  - 建议：100-200mg/天；和他汀分开 4 小时

### Q4

- **Q 原文**：妈妈在吃华法林抗凝，她朋友推荐的辅酶 Q10 能吃吗？
- **来源**：合理补充（家人场景 + 高风险药物）
- **场景类**：FR-3 + 家人场景 + 高风险药物
- **期望等级**：红色禁忌
- **VitaMe 输出概要**：
  - 红：辅酶 Q10 有 Vitamin K 样作用，可能降低华法林抗凝效果，增加血栓风险
  - 翻译：不是玄学，是药物抵抗机制
  - 建议：开始或停用前必须咨询医生并监测 INR

## B. 病史 × 补剂冲突（5 条）

### Q5

- **Q 原文**：家里老人有肝炎+脂肪肝，apoe4 基因，可以吃鱼油吗？什么样的鱼油？
- **来源**：真实评论（第 4 图，湖南 01-04）
- **场景类**：FR-3 + 家人场景 + 基因精准
- **期望等级**：黄色注意
- **VitaMe 输出概要**：
  - 黄：肝炎活动期需先看 AST/ALT；apoe4 对 DHA 代谢有差异
  - 追问：肝功当前值？凝血功能是否正常？（这是博主实际追问的，VitaMe 应自动化）
  - 建议：优先 EPA 为主（EPA:DHA ≥ 2:1）的鱼油；1g/天起；3 个月复查肝功

### Q6

- **Q 原文**：有肾结石，可以吃维 D 和钙片吗？维 C 呢？
- **来源**：合理补充（对应 PRD §8 用户故事 1）
- **场景类**：FR-3 + 病史交叉
- **期望等级**：黄色注意
- **VitaMe 输出概要**：
  - 黄：维 D 促钙吸收需控制总钙；维 C >1g/天增加草酸钙结石风险
  - 翻译：不是不能补，是要盯"剂量上限"
  - 建议：维 D 600-1000 IU/天；钙优先食补不另外补；维 C 不超过 500mg/天

### Q7

- **Q 原文**：胃溃疡体质，听说镁能助眠但容易腹泻，吃什么形式的？
- **来源**：合理补充（对应 0417 定位 Hero 示例"Doctor's Best 镁片"）
- **场景类**：FR-3 + FR-4（成分形式翻译）
- **期望等级**：黄色注意
- **VitaMe 输出概要**：
  - 黄：氧化镁吸收差且易致腹泻，胃肠敏感体质不合适
  - 翻译：镁形式差异 = 吸收率差异 + 刺激性差异
  - 建议：优先甘氨酸镁或苏糖酸镁，睡前 200mg

### Q8

- **Q 原文**：高血压+糖尿病，在吃二甲双胍和降压药，可以吃铬和肉桂粉吗？
- **来源**：合理补充（多重慢病场景）
- **场景类**：FR-3 + 多重慢病
- **期望等级**：黄色注意
- **VitaMe 输出概要**：
  - 黄：铬 / 肉桂有降糖作用，与二甲双胍叠加可能低血糖
  - 建议：起始低剂量 + 监测血糖 + 咨询医生调整用药

### Q9

- **Q 原文**：甲减在吃优甲乐，有人说不能和钙片一起吃？
- **来源**：合理补充（甲状腺高频场景）
- **场景类**：FR-3 + 时间间隔问题
- **期望等级**：黄色（非禁忌，时序问题）
- **VitaMe 输出概要**：
  - 黄：钙会抑制优甲乐吸收，需间隔 4 小时
  - 翻译：不是不能一起补，是"顺序"问题
  - 建议：优甲乐晨起空腹；钙片晚餐后

## C. 多补剂组合 × 时间安排（3 条）

### Q10

- **Q 原文**：鱼油、维 D、D3、镁、益生菌、维 B、维 C 都吃，每天什么时间吃？需要间隔多久？
- **来源**：真实评论（第 1 图，贵州 2025-08-21，博主回复 15 赞）
- **场景类**：FR-3 + FR-4（结构化时间表）
- **期望等级**：绿色（无明显冲突）
- **VitaMe 输出概要**：
  - 绿：大部分无需严格间隔
  - 结构化时间表：
    - 早空腹：维 B / 维 C / 益生菌（水溶性）
    - 早/晚餐随餐：维 D3 / 鱼油（脂溶性需脂肪协助吸收）
    - 睡前：镁（助眠）
  - 原因翻译：水溶性走尿代谢易排出；脂溶性需脂肪帮助吸收

### Q11

- **Q 原文**：一天吃 6 种补剂，分几次吃合适？能不能一把全吃？
- **来源**：合理补充
- **场景类**：FR-4（时间分配）
- **期望等级**：灰色（建议非强制）
- **VitaMe 输出概要**：
  - 灰：无明显禁忌，但分次效果更好
  - 建议：至少分 2 次（早/晚），避免脂溶性和高剂量钙镁同服

### Q12

- **Q 原文**：补剂要不要一起补？还是先补一种看效果再加下一种？
- **来源**：合理补充（决策辅助）
- **场景类**：FR-4 + 决策辅助
- **期望等级**：灰色（取决于目的）
- **VitaMe 输出概要**：
  - 灰：症状明确指向某种缺乏（如睡眠差→镁），先补单一 2-4 周；一般保健可组合起步
  - 建议：单一补剂 2-4 周观察反应，记录

## D. 食物/饮品 × 吸收影响（3 条）

### Q13

- **Q 原文**：早餐必喝咖啡，会影响随餐吃的补剂吸收吗？
- **来源**：真实评论（第 2 图，上海 2025-09-07）
- **场景类**：FR-4（吸收翻译）
- **期望等级**：黄色注意
- **VitaMe 输出概要**：
  - 黄：咖啡因和鞣酸会显著影响铁、钙、B 族吸收
  - 建议：含铁 / 钙 / B 族的补剂与咖啡间隔至少 1 小时
  - 脂溶性（维 D、鱼油）不受咖啡明显影响

### Q14

- **Q 原文**：钙片可以和牛奶一起吃吗？
- **来源**：合理补充
- **场景类**：FR-4
- **期望等级**：灰色（日常可接受）
- **VitaMe 输出概要**：
  - 灰：牛奶本身含钙，一起吃不"冲突"，但一次钙负荷 >500mg 吸收率显著下降
  - 建议：分次摄入更好

### Q15

- **Q 原文**：绿茶和铁剂可以一起吃吗？
- **来源**：合理补充
- **场景类**：FR-4
- **期望等级**：黄色注意
- **VitaMe 输出概要**：
  - 黄：茶多酚显著抑制非血红素铁（植物性铁+铁剂）吸收
  - 建议：间隔 2 小时

## E. 特殊场景（5 条）

### Q16

- **Q 原文**：甘氨酸镁、苏糖酸镁、氧化镁、柠檬酸镁，到底选哪个？
- **来源**：合理补充（0417 定位 Hero 首页示例核心场景）
- **场景类**：FR-4（成分形式翻译的代表场景）
- **期望等级**：N/A（决策辅助非风险判断）
- **VitaMe 输出概要**：
  - 形式对照表：
    - 氧化镁：便宜 / 吸收差 / 易腹泻
    - 柠檬酸镁：吸收较好 / 轻度通便（适合便秘）
    - 甘氨酸镁：温和 / 助眠 / 不腹泻
    - 苏糖酸镁：入脑 / 助认知 / 贵
  - 按目的推荐：睡眠→甘氨酸镁；便秘→柠檬酸镁；认知→苏糖酸镁

### Q17

- **Q 原文**：鱼油 EPA 和 DHA 比例怎么看？买哪种？
- **来源**：合理补充（第 4 图 hashtag #epa #omega3 已显示这是高频搜索）
- **场景类**：FR-4（成分形式翻译）
- **期望等级**：N/A
- **VitaMe 输出概要**：
  - 翻译：EPA 偏心血管 / 抗炎；DHA 偏脑认知 / 视力
  - 按目的：心血管 EPA:DHA=2:1 或纯 EPA；认知 1:2 或纯 DHA
  - 标签读法：看每粒总 Omega-3 含量 + EPA/DHA 细分 + 是否纯度 >60%

### Q18

- **Q 原文**：孕期维 D 要和非孕期一样剂量吗？
- **来源**：合理补充（孕妇高频场景）
- **场景类**：FR-3 + 特殊生理期
- **期望等级**：黄色（特殊群体需谨慎）
- **VitaMe 输出概要**：
  - 黄：孕期维 D 推荐 600-1000 IU/天，略高于一般成人但不要过量
  - 建议：结合 25(OH)D 水平调整；咨询产科医生
  - 强免责：孕期补剂必须经产科医生确认

### Q19

- **Q 原文**：这些补剂都可以一直吃吗？
- **来源**：真实评论（第 2 图，湖北 03-22）
- **场景类**：FR-4（服用周期翻译）
- **期望等级**：灰色（分类回答）
- **VitaMe 输出概要**：
  - 灰：分两类
    - 可长期：维 D（按剂量监测）、鱼油、益生菌
    - 不宜长期：高剂量铁、高剂量锌（>40mg/天需评估）、脂溶性维生素超量
  - 建议：3-6 个月复评血检

### Q20

- **Q 原文**：感觉全都要吃🥲，到底从哪个开始补？
- **来源**：真实评论（第 3 图，广东）
- **场景类**：首页引导 + 决策入口（非风险判断）
- **期望等级**：N/A
- **VitaMe 输出概要**：
  - 引导式回应：有没有明确症状（睡眠差 / 便秘 / 易疲劳）？有体检异常项吗？在吃什么药？
  - 按优先级推荐单一起始补剂（而非"都补"）
  - 这条是 P0 首页 Hero 下可放的"第一个决策辅助"钩子

## 使用路径

### 路径 A：Demo 脚本（90 秒）

**推荐组合 1（情绪+翻译+家人三合一）**：
- 开场：Q5 老人肝病+apoe4+鱼油（情绪重 + 精准 + 家人场景）
- 展开：输入后 VitaMe 自动追问凝血功能（呼应博主原本需要手动追问的内容）
- 输出：黄色 + EPA:DHA 比例建议 + 3 个月复查
- 收尾：保存到"妈妈档案" → 下次新增钙片直接复查

**推荐组合 2（翻译层亮点）**：
- 开场：Q16 四种镁形式
- 展开：展示翻译层"为什么形式会影响效果"
- 输出：按目的推荐（睡眠/便秘/认知）
- 收尾：保存为"我的睡眠方案"档案

**推荐组合 3（信息过载 → 引导）**：
- 开场：Q20"感觉全都要吃"
- 展开：VitaMe 首页引导式问答
- 输出：单一起始补剂推荐
- 收尾：用一个具体例子（Q10）展示"不确定什么时间吃？VitaMe 给表"

### 路径 B：规则引擎测试用例

每条 Q 的"期望等级"+"输出概要"→ 直接变 unit test 断言。

```
test("Q4 华法林+Q10 应触发红色禁忌", () => {
  const result = safetyJudgment({
    supplement: "辅酶Q10",
    medications: ["华法林"],
  });
  assertEqual(result.level, "red");
  assertContains(result.reason, "Vitamin K 样作用");
});
```

### 路径 C：LLM Prompt 验证集

20 条 Q 跑 DeepSeek prompt 后观察四个维度：
1. 色级是否正确
2. 红 / 黄场景是否强制附"咨询医生"
3. 翻译层文字是否普通人可懂（避免术语堆砌）
4. 免责声明是否自动附带

### 路径 D：用户访谈脚本

从 20 条里挑 3-5 条最贴近受访者真实场景的，让用户"用自己的话描述一次遇到这个问题时的流程"——验证 5.3 节的 4 个未验证假设。

## 一句话结论

**这 20 条种子问题同时服务 Demo 叙事、规则引擎冷启动、Prompt 验证、用户访谈四个目标，覆盖 P0 七个 FR 中的 5 个（80%+），剩余"档案复查"场景在家人问题中自然涵盖。**
