# VitaMe 竞品调研报告

> 调研日期：2026-04-16
> 产品定位：AI驱动的家庭保健品安全管理工具
> 核心差异化：药物 x 保健品 x 病史 三角交叉安全校验

---

## 六、竞争格局总结

### VitaMe的真正竞品是谁？

按威胁程度排序：

**第一梯队（功能直接重叠，需持续关注）：**
- **Drugs.com / WebMD Interaction Checker** -- 全球最大的药物交互检测工具，但是英文、无中文化、无个性化、无成分形式级分析、无家庭管理。VitaMe相当于把这类工具的核心能力做中国本土化+消费者友好化+个性化升级。
- **VitalStack / CanIStackThis** -- 新兴的AI补充剂交互检查工具，功能最接近VitaMe核心。但目前均为早期产品、仅英文、只做检测不做推荐和管理。需要关注它们是否会扩展功能。

**第二梯队（部分功能重叠，间接竞争）：**
- **Persona Nutrition** -- 唯一做药物-营养素交互筛查的推荐平台，但被雀巢收购后品牌锁定，只推自有产品。
- **Medisafe** -- 家庭用药管理标杆，但聚焦处方药，保健品覆盖弱。
- **蚂蚁阿福** -- 中国AI健康赛道绝对王者（3000万月活），但主动不碰保健品。既是潜在威胁（如果政策明确后它决定进入），也是VitaMe的市场验证（大厂不碰=创业机会）。

**第三梯队（赛道相邻，低直接竞争）：**
- LemonBox、Key科益家、Care/of等品牌锁定的个性化补充剂订阅服务
- 丁香医生、薄荷健康、Ada Health等健康科普/管理平台
- 用药助手等专业医护工具

### 蓝海判断：基本成立，但有结构性原因

**成立的理由：**
1. **中文市场"药物x保健品x病史"三角交叉安全校验工具 = 零竞品。** 这不是调研不够仔细，而是确实没有。
2. **大厂主动回避** -- 蚂蚁阿福3000万月活不碰，字节小荷不碰，说明这个空白不是大厂没看到，而是大厂因监管/责任风险选择不做。创业公司可以以更轻的身姿承担这个风险。
3. **全球范围拼图碎片散落** -- SUPP.AI有数据但非消费者产品，Persona做筛查但品牌锁定，Examine有知识但无个性化。VitaMe第一次把这些碎片拼到一个面向中国消费者的产品里。

**需要警惕的结构性挑战：**
1. **缝隙存在有原因。** 成分形式级分析需药学专业知识，补充剂交互数据库远不如处方药完善（尤其中国传统补品），品牌中立与变现天然矛盾。这些不是等着解决的问题，而是领域固有张力。
2. **防御壁垒有限。** 如果VitaMe模式被验证，蚂蚁或字节可以用压倒性分发能力复制。核心壁垒在于：(a)安全规则库的深度和医学审核质量，(b)用户家庭健康档案的数据积累和迁移成本，(c)先发信任品牌。
3. **监管是双刃剑。** 《广告法》第46条限制了"利用健康知识变相推荐保健品"，这既阻止了大厂进入，也约束了VitaMe的商业化路径。定位"安全检测"而非"保健品推荐"是正确的合规策略。

### 竞争定位建议

VitaMe应明确自身定位为：**家庭保健品安全管理工具**，而非"保健品推荐平台"或"AI健康顾问"。

- 与Drugs.com竞争维度：中文化 + 个性化 + 消费者友好
- 与蚂蚁阿福的关系：互补而非竞争（做阿福不做的事）
- 与品牌锁定玩家的差异：品牌中立 + 安全优先
- 核心护城河：医学博士审核的规则库 + 家庭健康档案数据飞轮
